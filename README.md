# KalaGraphics

Shader, texture, UI and model handling in OpenGL 3.3, with plans for Vulkan 1.3. Should theoretically work just fine with GLFW or anything that provides a GL context and can render a window, but recommended to be used with [KalaWindow](https://github.com/kalakit/kalawindow)
